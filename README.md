### Задание 1

**1.1. Бюджетирование, финансовая отчётность, прогнозирование рисков**

Задача: Требуются гарантии целостности данных (ACID), чёткая схема и мощные возможности для аналитики.

**Выбранный тип СУБД**: Реляционная СУБД (SQL) - Oracle, MySQL, PostgreSQL (импортозамещенный аналог)

Реляционные СУБД основаны на ACID-транзакциях (Atomicity, Consistency, Isolation, Durability), что гарантирует точность и надежность финансовых данных. Строгая схема данных (предопределенные таблицы, столбцы, типы данных, связи) исключает аномалии и обеспечивает четкую структуру, необходимую для бухгалтерского учета и отчетности. Поддерживает интеграцию с BI-инструментами (Power BI, Tableau).`
